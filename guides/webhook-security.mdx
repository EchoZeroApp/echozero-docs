---
title: Webhook security
description: Verify inbound provider webhooks with shared secrets, HMAC, and replay protection.
---

Inbound webhooks (Telegram signal bots, Discord gateways, agent HTTP ingress) must be verified before Echo Zero processes payloads.

## Practices

- **Shared secret headers** — e.g. `X-Telegram-Bot-Api-Secret-Token` when configured for Telegram bots.
- **HMAC signatures** — validate `x-signature` + `x-timestamp` on agent inbound HTTP URLs.
- **Replay protection** — reject timestamps outside the allowed drift window and deduplicate by event id.
